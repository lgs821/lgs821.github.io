TortoiseGit+MsysGit详细安装教程-代码托管工具不二之选

@IMxcz.Com · 2014/12/25 · 3 分钟 · 软件

说到代码托管，熟悉的朋友首先就会想到Github这货了吧，全世界的程序员都了解的东西，当然Svn也是不错的选择。使用代码托管当然需要好用的管理工具，无疑TortoiseGit + MsysGit是很方便的Windows平台下再方便不过的Git客户端工具了，在Mac下推荐使用Sourcetree，当然由于小屌丝一枚，没有苹果这么高大上的电脑，只要就说说Windows下的安装使用说明吧。0.0

首先当然是需要下载软件了

TortoiseGit下载地址：https://code.google.com/p/tortoisegit

MsysGit下载地址：http://msysgit.github.com/ ( Git for Windows )

当然国外的服务器下载速度是很让人郁闷的，不想受这煎熬的同学可以在这里下载

http://pan.baidu.com/s/1gdy5psR解压密码：www.imxcz.com

版本分别是TortoiseGit1.8.12 32位和64位 包含中英文语言包 MsysGit1.9.5 都是2014年圣诞节前后的最新版本，如果想使用更新版本可以随时关注它们官方网站自行下载即可。

准备工作完成，接下来一步步安装

首先安装MsysGit-Git for Windows,安装到您需要的位置即可

接着安装TortoiseGit根据提示直接下一步到安装结束。安装完成后点击右键可以看到图标了，点击Setting，如图所示：

打开后在General页面点击Check now左边会显示Git的版本号，说明安装已经成功了。那接下来呢，接下来当然是安装语言包了，对于英文不好的童鞋，当然还是简体中文的有木有！找到下载的压缩包里对应系统类型的语言包直接双击安装，很快就安装好，然后点开设置，还是如上图Language那里选择 中文-简体(中华人民共和国) 点击之后有木有感觉界面一下子很亲切了呢。o(^▽^)o

接下来创建版本库

在项目中空白处点击鼠标右键，安装过语言包的同学会看到 Git 在这里创建版本库 的选项点击，然后点击弹出框的“确定”，项目中会出现一个隐藏的 .gif文件夹，说明创建成功。

接下来设置版本库，项目中点击右键->TortoiseGit->设置 然后点击Git,填写名字，Email，接着点击 远端->填写远端名称，Url那里填写Git的https地址或者SSH地址都可，然后点击添加保存，就完事儿了。

接下来就是软件的使用了

使用很简单的，大致说一下，就不详细说了，自己慢慢摸索就会了。

项目代码写好了，或者修改了功能之后在项目目录右键->Git提交

在日志信息那里填写修改或者新增的功能之类的说明，下面选择需要提交的文件，点级确定。进度完成显示成功后，点击 推送。

选择本地和远端的地址，确定。弹出框输入Git的账号，密码等待成功即可。。

到此结束。其他功能不熟悉的自己摸索哦。。。。
