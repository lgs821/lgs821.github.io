Mac OS下Golang环境配置详细说明

@IMxcz.Com · 2017/03/17 · 2 分钟 · 技术文章

golang是什么语言，相信大多数人都有一个概念了，实在不知道，自行百度吧，go语言还是比较新的语言，发布短短几年，使用者也是相当可观，非常适合服务端程序的开发。golang是可以跨平台的，这里就简单说说在max os下Go语言环境的配置，当然，配置方法有很多种，可以用brew快速安装，这里只说从官网下载，手动安装配置的方法。

首先，需要去官网下载最新版本，官网地址,当然，众所周知的原因，国内访问仍然需要梯子，这里提供下百度网盘的链接（虽然不是会员，下载速度你会非常郁闷），可以在这里下载，版本号：go1.8 64位

下载地址：百度网盘

下载后是一个pkg的安装包，直接打开安装一路继续不要停直到安装完成：

接着就是配置环境变量，打开终端输入：

终端中输入

vim .bash_profile

在新窗口中按 “i” 键进入编辑状态，输入活着复制粘贴下面代码：

环境变量

export GOPATH=/Users/lishuai/Documents/GoPath   //这里请根据情况设置为自己项目地址
export GOBIN=$GOPATH/bin
export PATH=$PATH:$GOBIN

接着按 “ESC” 键退出编辑状态，输入 “:wq” 保存退出，想要即时生效，输入以下代码，回车：

刷新环境变量

source .bash_profile

至此环境变量已经配置结束了。终端输入：go env 查看

接下来验证Golang是否能正常运行

在\$GOPATH文件夹下心间文件夹 src/hello。在​$GOPATH/src/hello 下心间hello.go文件，内容如下：

package main
 
import "fmt"
 
func main(){
	fmt.Printf("hello,www.imxcz.com\n");
}
